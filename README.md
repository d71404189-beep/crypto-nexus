# crypto-nexus
Secure multi-chain wallet for BTC, ETH, BNB, SOL, TRX. Create or import wallet, view real balances.
How to Install Crypto Nexus Wallet
Download the ZIP archive from the link we provided.

Extract the ZIP file to any folder on your computer.

Open Google Chrome and type chrome://extensions/ in the address bar.

Enable Developer Mode using the toggle switch in the top‑right corner.

Click “Load unpacked” and select the folder where you extracted the files.

The wallet will appear in your extensions list – you’re ready to go!
